git init (把這個資料夾變成repo)
git add . (把所有資料夾路徑內容加入)
git add "file name.format" (只把這個檔案加入)
git status (確認目前狀態)
git commit -m "comment" (儲存的動作，""內是上傳到github的註解)
git remote (確認目前remote對應到的remote)
git remote -v (目前github and git 之間對應路徑)
git remote add origin https://github.com/mnihank/MapleStory.git (範例)
git remote set-url origin "http:/" (更改與github連接之路徑)
git log (執行紀錄)
git clone "http:/" (下載)
git push -u "remote name" "branch" (上傳至github)
